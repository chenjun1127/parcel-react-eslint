### Parcel-React-Eslint

Parcel 配置的 react 项目，配置了 eslint+sass+css-module+prettier 。

### 常用参数

* -p, –port 设置服务器端口
* –https 在https协议上运行
* -o, –open 自动在默认浏览器打开
* -d, –out-dir 设置输入路径默认为dist
* –public-url 设置服务器运行的路劲. 默认与–out-dir option 设置的相同
* –no-hmr 关闭热模块替换
* –no-cache 关闭缓存
* -V, –version 版本信息
* -h, –help 帮助信息
